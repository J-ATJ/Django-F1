Django F1 App Preview

This project is a web application built with Django that uses webscraping to get the current data from the 2023 Formula 1 championship.

The main view shows the leaderboard of the top 3 drivers with their photos and relevant data such as name, team, country and points won so far. In addition, it shows the complete tables of the drivers' and constructors' championships with the total points and the position obtained in each race. It also shows details of the upcoming Grand Prix, including the date, name and a countdown to the race.

Data was retrieved via webscraping from different sources using the Python "requests" and "beautifulsoup" libraries.

The app is responsive and has a simple design using bootstrap 5.
